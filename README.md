Financial Insight – Intelligent Platform for Monitoring and Analysis of Personal Economic Indicators

Setup
1.	Download or copy the project into a new folder. 
2.	Make sure the files main.html, style.css, script.js, and dashboard.js are located in the same directory. 
3.	Open the main.html file in a web browser to run the application.

Description
Financial Insight is a web application designed for managing personal income and expenses. The platform allows users to enter financial data, organize transactions by categories, and track totals through a simple and intuitive interface.
The application is developed using HTML, CSS, and JavaScript and serves as a foundation for future extensions such as charts, statistics, and permanent data storage.
Important things for people trying the project
•	The project files must remain in the same folder for the application to function correctly. 
•	The browser must have JavaScript enabled. 
•	No server or additional software installation is required to run the application.

Instructions on How the Site Works
1.	Choose the desired mode: 
      Expenses for adding expenses 
      Income for adding income 
2.	Enter the amount in the Amount field. 
3.	Select a category from the dropdown menu. 
4.	Optionally, add a short description for the transaction. 
5.	Press the Add button to save the entry. 
6.	The transaction will appear in the list below together with: 
      category 
      amount 
      description 
      date 
7.	The application automatically calculates and displays the total amount for the selected mode. 
8.	Switching between Expenses and Income changes the displayed categories, entries, and totals.
